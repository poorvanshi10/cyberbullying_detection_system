# 🛡️ SafeNet: Cyber Bullying Detection System

> An interactive, visually engaging frontend for an AI-powered toxicity detection engine.

## 🎯 Project Overview
SafeNet is a frontend architecture designed to demonstrate how artificial intelligence can be used to combat cyberbullying. It features a modern glassmorphism interface and a reactive 3D background to simulate a live digital network. The ultimate goal of this project is to show how real-time NLP (Natural Language Processing) can identify harmful language and foster safer online communities.

## ✨ Key Features
* **Simulated AI Analysis:** Type a message and watch the system instantly evaluate its toxicity and provide contextual feedback.
* **Interactive 3D Web:** A dynamic, mouse-reactive particle background built with Three.js that visually represents the digital ecosystem.
* **Premium UI/UX:** A sleek, responsive, dark-themed interface built entirely with Tailwind CSS for a premium feel.
* **Instant Performance:** Runs entirely in the browser with zero latency, providing an immediate feedback loop.

## 💻 Tech Stack
* **Core:** HTML5, Vanilla JavaScript
* **Styling:** Tailwind CSS
* **3D Graphics:** Three.js
* **Iconography:** Lucide Icons

## 🚀 How to Run Locally
This project requires no complex setups or package managers. 
1. Clone this repository or download the files.
2. Open the project folder.
3. Double-click the `index.html` file to launch it in your preferred web browser.

## 🔮 Future Scope (Backend Integration)
While this repository contains the frontend interface and a simulated response engine, the architecture is structured so it can easily connect to a fully functional machine learning backend (e.g., Python, TensorFlow, and FastAPI) for real-world NLP processing in the future.
